# ReCode - AI-Powered Code Analyzer Chrome Extension 🚀

## 🌟 Overview

**ReCode** is a cutting-edge AI-powered Chrome extension designed to analyze your code directly from coding platforms or manual input. It provides real-time complexity analysis, code quality feedback, alternative solutions, test case generation, and hints, all within a ChatGPT-style interactive interface.

Built for developers, competitive programmers, and students, ReCode helps optimize, understand, and improve code efficiently.

## ✨ Key Features
*	**AI Code Analysis**: Leverages Gemini API to deeply analyze code.
*	**ReCode Hybrid Suggestions**: Combines AI insights with local logic for actionable recommendations.
*   **Time & Space Complexity**: Automatically predicts complexities with visual graphs.
*	**ChatGPT-Style UI**: Interactive conversation-like bubbles for user input and AI output.
*	**Dynamic Typing Animation**: AI responses appear letter-by-letter for a realistic feel.
*	**Manual Code Input**: Paste code directly if auto-extraction fails.
*	**Extracted Code Viewer**: View code extracted from supported coding websites.
*	**Customizable Theme**: Toggle between light and dark modes.
*	**Clipboard Integration**: Copy AI responses and complexity metrics easily.
*	**Real-time Graphs**: Visualize code complexity trends instantly.

## 🎯 Supported Platforms
*	LeetCode
*	GeeksforGeeks
*	CodeChef
*	Other coding platforms with visible code blocks

## 🛠 Installation
*	Clone the repository: git clone https://github.com/nehamalik12210/leetcode-solution-extension.git
*  Open Chrome and navigate to chrome://extensions/.
*	Enable Developer Mode.
*	Click Load unpacked and select the recode folder (project folder).
*	The extension icon will appear in the Chrome toolbar.
	
## 📖 Usage
*	Open any coding problem page.
*	Click the ReCode extension icon.
*	Code will be automatically extracted, or use manual input if needed.
*	Choose an action:
*	Analyze Complexity
*	Deep Explain
*	Code Review
*	Generate Tests
*	Better Approaches
*	Smart Hints
*	View results in the ChatGPT-style interface with AI typing animation and complexity graphs.
